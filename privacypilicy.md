# Privacy Policy

**Application:** OKAS SIP Door Phone

## 1. Introduction
OKAS SIP Door Phone is a SIP-based door phone and communication application.

This Privacy Policy explains how we collect, use, transmit, and protect information when you use our application.

By installing or using the application, you agree to the practices described in this Privacy Policy.

---

## 2. Information We Collect

### 2.1 SIP Account Information
- SIP username
- SIP password
- SIP server domain or IP address
- Display name

Stored locally. Not sold or shared for advertising.

### 2.2 Audio Data (Microphone Permission – RECORD_AUDIO)
Used only during active SIP calls. Routed through configured SIP server.

### 2.3 Video Data (Camera Permission – CAMERA)
Used for video calls. Not stored unless configured by user.

### 2.4 Network Information
Used for SIP registration and communication stability.

### 2.5 Notifications
Used for incoming call alerts and call status updates.

### 2.6 Call Management
Used for handling SIP calls only.

### 2.7 Background Services
Used to maintain SIP connectivity.

### 2.8 Wake & Boot Services
Used to maintain communication reliability.
