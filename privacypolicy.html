Privacy Policy

1. Introduction

OKAS SIP Door Phone is a SIP-based door phone and communication application.

This Privacy Policy explains how we collect, use, transmit, and protect information when you use our application.

By installing or using the application, you agree to the practices described in this Privacy Policy.

2. Information We Collect

2.1 SIP Account Information

When configuring the application, users may enter:

  • SIP username
  • SIP password
  • SIP server domain or IP address
  • Display name

This information is used solely to enable SIP-based voice and video communication.

SIP credentials:

  • Are stored locally on the user’s device
  • Are not transmitted to our company servers unless the SIP server is operated by your organization
  • Are not sold, rented, or shared for advertising or marketing purposes

2.2 Audio Data (Microphone Permission – RECORD_AUDIO)

The application requests microphone access to:

  • Enable two-way voice communication
  • Support SIP audio calls
  • Allow door phone interaction

Audio data:

  • Is transmitted only during active communication sessions
  • Is routed through the configured SIP server
  • Is not recorded or stored by us unless the user explicitly enables recording functionality

We do not sell or use audio data for advertising.

2.3 Video Data (Camera Permission – CAMERA)

The application may request camera access to:

  • Enable SIP video calls
  • Display live video streams from connected SIP door devices

Video data:

  • Is transmitted only during active communication sessions
  • Is routed through the configured SIP server
  • Is not stored or archived by us unless explicitly configured by the user

2.4 Network and Connectivity Information

The application may process technical information including:

  • IP address
  • Network connection status
  • Wi-Fi connection state
  • Device connectivity status

This information is required to:

  • Maintain SIP registration
  • Receive incoming calls
  • Ensure stable communication
  • Restore service after device reboot

This information is used strictly for communication functionality and not for tracking or advertising.

2.5 Notifications (POST_NOTIFICATIONS, USE_FULL_SCREEN_INTENT)

The application uses notifications to:

  • Inform users of incoming calls
  • Display missed call alerts
  • Provide call status updates

Full-screen notifications may be used during incoming calls to provide a call-like user experience.

Notification data is used solely for communication functionality.

2.6 Call Management (MANAGE_OWN_CALLS, USE_SIP)

The application uses Android call management features to:

  • Handle SIP calls
  • Display call UI
  • Manage call states (incoming, ongoing, ended)

The app does not access or manage cellular carrier call data.

2.7 Background Services & Foreground Services

The application may use foreground services and background processing to:

  • Maintain active SIP registration
  • Support real-time incoming calls
  • Enable microphone and camera during calls
  • Maintain reliable communication sessions
  • Perform data synchronization required for SIP connectivity

Foreground services used include:

  • Phone call service
  • Microphone service
  • Camera service
  • Data sync service
  • Special use service (for full-screen call notifications)

These services are used exclusively for communication functionality and not for tracking or advertising.

2.8 Device Wake & Boot Services (WAKE_LOCK, RECEIVE_BOOT_COMPLETED)

The application may:

  • Prevent device sleep during active calls
  • Restart SIP registration after device reboot

These features ensure reliable communication functionality.

No additional personal data is collected during these processes.

3. Third-Party SIP Server Processing

OKAS SIP Door Phone enables communication through a SIP server that may be:

  • Operated by your organization, or
  • Operated by a third-party SIP service provider

When you use the app:

  • Call signaling data is transmitted to the configured SIP server
  • Audio and video streams are routed through the SIP server

We do not control the privacy practices of independently operated third-party SIP servers.

Users are responsible for reviewing the privacy policies of their SIP service provider.

We do not sell or commercially exploit communication data.

4. Data Storage & Retention

  • SIP credentials are stored locally on the device
  • We do not centrally store voice content
  • We do not centrally store video content
  • We do not maintain centralized call recording storage
  • Call metadata (if generated) is handled by the configured SIP server

Users may delete local data by:

  • Logging out
  • Clearing app data
  • Uninstalling the application

5. Data Sharing

We do not sell, rent, or trade personal data.

Information may be transmitted only to:

  • The configured SIP server for communication routing
  • Network providers necessary to establish connectivity

We do not share data for advertising or marketing purposes.

6. Data Security

We implement reasonable technical and organizational safeguards including:

  • Secure credential storage on device
  • Encrypted SIP communication (if supported by the SIP server)
  • Restricted background processing
  • Controlled permission access

However, no transmission method over the Internet is completely secure.

7. Legal Basis for Processing (Where Applicable)

We process data to:

  • Provide communication services requested by the user
  • Enable SIP functionality
  • Maintain service reliability

By using the application, you consent to this processing.

8. Children’s Privacy

This application is not directed to children under 13 years of age.

We do not knowingly collect personal information from children.

9. Third-Party Libraries & Services

The application may use:

  • Android system services
  • SIP communication libraries (such as Linphone or similar libraries)
  • Push notification services (if enabled)

These third-party components may process technical data according to their respective privacy policies.

10. International Data Transfers

If the configured SIP server is located outside your country, communication data may be transmitted internationally as required for communication functionality.

By using the application, you consent to such transfers.
